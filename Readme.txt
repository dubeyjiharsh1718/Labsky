# Project Name: Test Assignment - HTML, CSS, Bootstrap

## Description:
This project serves as a test assignment where HTML, CSS, and Bootstrap have been utilized to create a responsive and visually appealing web page.

## Installation:
To run this project locally, follow these steps:

1. Clone the repository to your local machine.

2. Open the project directory.

3. Open the `index.html` file in your web browser.

## Usage:
This web page showcases the application of HTML for structure, CSS for styling, and Bootstrap for responsive design. Feel free to explore the code and make any necessary modifications.
